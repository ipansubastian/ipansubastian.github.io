Memahami Sintaks Markdown

Markdown merupakan bahasa markup yang lebih sederhana daripada HTML. Markdown sendiri sering dipakai untuk memformat teks yang tidak terlalu kompleks, tapi butuh pemformatan yang dapat memberi sedikit Style pada file tersebut. Sintaks Markdown sendiri sangat sederhana, dan tag-tag Markdown tidak sebanyak tag HTML. Dan biasanya digunakan untuk membuat File Readme.md dalam sebuah project Open Source. Jika anda menemukan file teks berekstensi .md bisa dipastikan itu adalah file Markdown. Baik, mari kita mengenal Markdown lebih dalam dan mempelajari Sintaks-nya.
