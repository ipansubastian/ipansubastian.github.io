Memahami Sintaks Markdown

Markdown merupakan bahasa markup yang lebih sederhana daripada HTML. Markdown sendiri sering dipakai untuk memformat teks yang tidak terlalu kompleks, tapi butuh pemformatan yang dapat memberi sedikit Style pada file tersebut.

Sintaks Markdown sendiri sangat sederhana, dan tag-tag Markdown tidak sebanyak tag HTML. Dan biasanya digunakan untuk membuat File Readme.md dalam sebuah project Open Source. Jika anda menemukan file teks berekstensi .md bisa dipastikan itu adalah file Markdown.

Baik, mari kita mengenal Markdown lebih dalam dan mempelajari Sintaks-nya.

Membuat Heading / Penjudulan

Untuk membuat heading kita bisa menulis sebagai berikut.

# Judul 1

## Judul 2

### Judul 3

#### Judul 4

##### Judul 5

###### Judul 6

Hasil akhir akan menjadi seperti ini.

Judul 1

Judul 2

Judul 3

Judul 4

Judul 5
Judul 6

List

Untuk membuat list ada dua macam. Yaitu list tidak berurutan dan list berurutan.

Cara membuat list tak berurutan seperti ini.

* induk satu

* induk dua

  * anak satu dari induk dua

  * anak dua dari induk dua

Hasil tampilan diatas seperti dibawah ini.

  • induk satu
  • induk dua
    • anak satu dari induk dua
    • anak dua dari induk dua

Selanjutnya, cara membuat list berurutan.

1. induk satu

2. induk dua

3. induk tiga

   1. anak satu induk tiga
   
   2. anak dua induk tiga

Tampilan akhir akan seperti ini.

  1. induk satu
  2. induk dua
  3. induk tiga
    1. anak satu induk tiga
    2. anak dua induk tiga

Gambar

Untuk menginput suatu gambar kita bisa membuat kode seperti dibawah ini.

![Tori](https://raw.githubusercontent.com/zyddv/Web-Assets-Storage/master/shashin/nature-bird-red-wildlife-36762.jpg)

Kode diatas akan menampilkan gambar yang saya simpan di portal hosting repositori GitHub. Format yang digunakan adalah ![Teks Alternatif](URL Gambar).

Teks alternatif akan tampil saat gambar yang dipanggil tidak ditemukan.

Dan jika kita menjalankan kode diatas dengan Markdown Reader akan tampil gambar seperti berikut.

Link

Untu membuat link hampir mirip seperti membuat gambar, tapi tidak diawali dengan tanda seru. Berikut contohnya.

[My Facebook Account](https://m.facebook.com/ipansubastian.id)

Hasilnya akan tampil teks hyperlink seperti dibawah.

My Facebook Account

Blockquotes

Jika di HTML kita mengenal tag blockquotes untuk membuat quote / kata mutiara, di Markdown kita menggunakan cara seperti ini.

Mario Teguh:
> Keuntungan dan kerugian adalah dua sisi dari satu koin yang sama. Terlalu takut rugi sama dengan menjauhi
keuntungan.

Kode diatas akan menampilkan teks dengan format blockquotes.

Mario Teguh:

Keuntungan dan kerugian adalah dua sisi dari satu koin yang sama. Terlalu takut rugi sama dengan menjauhi keuntungan.

Penekanan

Terus gimana cara buat penekanan / teks miring dan tebal? Nah begini caranya.

Teks miring / italic

*teks ini akan miring*

_teks ini juga miring_

Hasilnya gimana? Seperti dibawah ini.

teks ini akan miring

teks ini juga miring

Teks tebal / bold

**teks ini akan tebal**

__teks ini juga tebal__

Markdown akan memformat teks diatas menjadi seperti ini.

teks ini akan tebal

teks ini juga tebal

Karakter Escape

Untuk menampilkan karakter khusus di dalam Markdown kita bisa menggunakan karakter escape backslash ‘\’.

Contohnya untuk karakter * dan _ yang digunakan untuk membuat teks menjadi miring, dengan karakter backslash karakter-karakter itu tak akan lagi dianggap khusus. Perhatikan contoh berikut.

Jika saya ingin menulis *emphasis* di Markdown akan dihasilkan teks miring emphasis. Agar teks tersebut tetap tampil *emphasis* kita bisa merubahnya menjadi *\emphasis\*. Maka teks yang dihasilkan adalah: emphasis.

Karakter Backslash dapat digunakan untuk mengescape karakter-karakter berikut :

  • ()
  • .
  • #
  • +
  • –
  • *
  • _
  • `
  • {}
  • !
  • []
