Tools Untuk Belajar Javascript di Smartphone Android

Sistem Operasi Android bisa dibilang Sistem Operasi mobile yang canggih, dan di toko aplikasi milik Android sendiri yaitu Google Play Store telah tersedia berbagai aplikasi premium. Namun tidak untuk keperluan programming, Aplikasi pendukung untuk programming bisa dibilang masih kurang.

Aplikasi-aplikasi untuk keperluan programming seperti text editor, IDE, compiler dan sebagainya belum tercukupi atau kurang ketersediaan di Google Play Store. Tapi meski begitu masih ada alternatif-alternatif lain sebagai jalan keluar jika anda ingin menulis kode di Smartphone Android anda. Terlebih untuk pemrograman berbasis web, alat-alat yang dibutuhkan juga tidak terlalu sulit untuk didapat.

Di artikel kali ini saya akan membeberkan tools atau alat-alat apa saja yang dibutuhkan untuk coding JavaScript di Android.

Acode â€“ powerful text/code editor

Alat yang pertama yang dapat anda gunakan adalah Acode, menurut saya sejauh ini inilah code editor yang paling bagus untuk JavaScript di Android. Acode dilengkapi dengan code completion yang tersedia untuk JavaScript, dan juga dilengkapi browser built-in yang dapat anda pakai untuk menampilkan hasil source code tanpa membuka browser. Selain itu anda juga bisa berinteraksi secara langsung dengan konsol JavaScript yang disediakan oleh Acode. Tersedia juga fitur Syntax Highlighting untuk memudahkan kita dalam membaca kode. Karena fitur-fiturnya ini, Acode lebih mirip seperti sebuah IDE untuk Web Development dengan fitur minimal ketimbang sebuah teks / code editor.

Acode juga mendukung bahasa-bahasa lain diluar Web Development, dan juga mendukung saran keyword & syntax highlighting di bahasa-bahasa itu.

Anda dapat mendownload Acode disini :

https://play.google.com/store/apps/details?id=com.foxdebug.acodefree

JSide

Opsi selanjutnya yang dapat anda pilih adalah JSide. Sama seperti Acode, ini merupakan code editor untuk JavaScript yang juga dilengkapi dengan konsol JavaScript.

Salah satu kekurangan JSide adalah dia tidak bisa menjalankan file HTML, melainkan hanya JavaScript saja.

Anda dapat mendownload JSide pada link dibawah ini :

https://play.google.com/store/apps/details?id=net.nizarmah.jside

Node.js di Termux

Alat ketiga ini merupakan alat untuk menjalankan JavaScript di sisi server. Ini dimungkinkan dengan adanya paket Node.js yang terinstall pada aplikasi Termux di Smartphone Android anda. Salah satu kekurangannya tidak bisa mengakses objek-objek yang terdapat di JavaScript di sisi browser, karena memang objek-objek itu hanya melekat di web browser. Untuk installasinya sendiri pertama anda harus menginstall aplikasi Termux di Smartphone anda. Anda bisa mendapatkannya di : https://play.google.com/store/apps/details?id=com.termux.

Setelah terinstall anda ketikkan perintah dibawah ini ( pastikan koneksi internet anda berjalan dengan baik ) :

pkg update && pkg upgrade;
pkg install nodejs -y

Setelah proses instalasi selesai, ketikkan node dan anda akan masuk ke konsol JavaScript. Jika anda ingin menjalankan kode JavaScript dalam sebuah file, ketik saja :

node nama_file.js

Maka kode anda akan diinterpretasikan oleh NodeJS.
